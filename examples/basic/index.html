SQLite3 IndexedDB VFS Example

This example demonstrates how to use the SQLite3 IndexedDB VFS in a WebAssembly environment.

Output:

How it works:

This example uses the SQLite3 IndexedDB VFS to store a SQLite database in the browser's IndexedDB. The database persists across browser sessions, so you can close and reopen the page and your data will still be there.

Steps to run this example:

  1. Compile the Go code to WebAssembly: GOOS=js GOARCH=wasm go build -o main.wasm
  2. Copy the WebAssembly support file: cp "$(go env GOROOT)/misc/wasm/wasm_exec.js" .
  3. Serve this directory with a web server: go run server.go
  4. Open http://localhost:8080 in your browser
